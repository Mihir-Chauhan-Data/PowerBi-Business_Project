** What this project is about

This dashboard helps to:

see how many patients visit the emergency room

track how long patients wait

understand patient age and gender distribution

check which departments receive the most referrals

review patient experience through ratings

It is designed for quick and clear decision-making.

**Main numbers shown on the dashboard

Total number of patients

Average waiting time

Average patient rating

Admitted vs not admitted patients

Patients attended on time vs delayed
** What you can learn from this dashboard
Patient overview

Male and female patient distribution

Patient count by different age groups

Service performance

How many patients were attended on time

How many experienced delays

Overall average waiting time

Patient experience

Average satisfaction rating

Department referrals

You can clearly see how many patients were referred to:

General Practice

Orthopedics

Cardiology

Neurology

Gastroenterology

Physiotherapy

Renal

Time analysis

Monthly view (January to December)

Year comparison for 2023 and 2024

**Interactive features

Year filter (2023 and 2024)

Month filter (January to December)

All charts update automatically when you select any filter

**Tools used

Power BI Desktop

DAX measures for KPIs

Data modeling and relationships

Interactive visuals and slicers

** About the data

The dataset includes:

Patient ID

Gender

Age and age group

Waiting time

Attendance status (on time or delayed)

Department referral

Admission status

Patient satisfaction score

Visit date

** Who can use this dashboard

This dashboard is useful for:

hospital operations teams

healthcare data analysts

management teams

It helps them monitor performance and improve patient services.
